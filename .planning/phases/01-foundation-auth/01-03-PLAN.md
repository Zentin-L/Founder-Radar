## Task: Implement auth state management & route protection

**API client** (`/lib/api.ts`):
- Axios instance with baseURL pointing to FastAPI
- `withCredentials: true` for cookie-based auth
- Response interceptor: on 401, try refresh; if refresh fails, redirect to /login

**Auth hooks** (`/lib/auth.ts`):
- `useAuth()` hook using TanStack Query
  - Calls GET /api/auth/me
  - Returns `{user, isLoading, isAuthenticated, login, register, logout}`
- `login(email, password)` — POST /api/auth/login, invalidate queries
- `register(email, password)` — POST /api/auth/register
- `logout()` — POST /api/auth/logout, redirect to /login

**Middleware** (`middleware.ts`):
- Check for auth cookie on protected routes (`/feed`, `/explore`, `/watchlist`, `/settings`)
- Redirect to `/login` if no cookie
- Allow public routes: `/`, `/login`, `/register`, `/reset-password`

**Email verify banner** component:
- Shows "Verify your email" banner at top of dashboard if `email_verified` is false
- Dismissible but reappears on next visit
</task>
