## Task: Initialize monorepo structure

Create the root project structure with `/frontend` and `/backend` directories:

```
founder-radar/
├── frontend/           # Next.js 15 app
├── backend/            # Python/FastAPI app
├── docker-compose.yml  # PostgreSQL + Redis
├── .gitignore
├── .env.example
└── README.md
```

Create `.gitignore` with Python, Node, and IDE entries.
Create `.env.example` with all required environment variables:
- DATABASE_URL, REDIS_URL
- JWT_SECRET, JWT_ALGORITHM=HS256, ACCESS_TOKEN_EXPIRE_MINUTES=30, REFRESH_TOKEN_EXPIRE_DAYS=30
- FRONTEND_URL (for CORS)
</task>

<task id="1.2">
## Task: Set up Docker Compose for PostgreSQL + Redis

Create `docker-compose.yml` with:
- PostgreSQL 16 on port 5432 with named volume for persistence
- Redis 7 on port 6379
- Default credentials matching .env.example

Verify both services start with `docker-compose up -d`.
</task>

## Task: Initialize FastAPI backend

Set up Python backend at `/backend`:

```
backend/
├── app/
│   ├── __init__.py
│   ├── main.py          # FastAPI app, CORS config, router includes
│   ├── config.py        # Pydantic settings from .env
│   ├── database.py      # Async SQLAlchemy engine + session
│   ├── auth/
│   │   ├── __init__.py
│   │   ├── models.py    # User model
│   │   ├── schemas.py   # Pydantic request/response
│   │   ├── routes.py    # Auth endpoints (placeholder)
│   │   └── services.py  # Auth business logic (placeholder)
│   ├── startups/
│   │   ├── __init__.py
│   │   └── models.py    # Startup model
│   └── signals/
│       ├── __init__.py
│       └── models.py    # Signal model
├── alembic/
│   ├── env.py
│   └── versions/
├── alembic.ini
├── requirements.txt
└── .env
```

Dependencies: fastapi, uvicorn, sqlalchemy[asyncio], asyncpg, alembic, python-jose[cryptography], passlib[bcrypt], pydantic-settings, slowapi, python-multipart

FastAPI app with CORS configured for frontend origin.

Database models:
- **User**: id (UUID), email (unique), password_hash, email_verified (bool), subscription_tier (enum: free/pro), sector_preferences (JSONB), created_at, updated_at
- **Startup**: id (UUID), name, domain, sector, stage, founded_year, description, team_size, momentum_score (float), score_updated_at, created_at, updated_at
- **Signal**: id (UUID), startup_id (FK), signal_type (enum: hiring/linkedin), value (float), previous_value (float), delta (float), collected_at, created_at

Create initial Alembic migration and run it to create tables.
</task>

<task id="1.4">
## Task: Initialize Next.js frontend

Set up Next.js 15 at `/frontend`:

```bash
cd frontend
npx -y create-next-app@latest ./ --typescript --tailwind --eslint --app --src-dir --no-import-alias
```

Install additional dependencies:
- `@tanstack/react-query` for server state
- `axios` for API calls

Create basic app structure:
```
frontend/
├── src/
│   ├── app/
│   │   ├── layout.tsx       # Root layout with QueryProvider
│   │   ├── page.tsx         # Landing/home page (placeholder)
│   │   ├── (auth)/
│   │   │   ├── login/page.tsx
│   │   │   └── register/page.tsx
│   │   └── (dashboard)/
│   │       └── feed/page.tsx  # Signal feed (placeholder)
│   ├── lib/
│   │   ├── api.ts           # Axios instance pointing to FastAPI
│   │   └── providers.tsx    # React Query provider
│   └── middleware.ts        # Route protection (check auth cookie)
```

Verify Next.js boots at `http://localhost:3000`.
</task>
