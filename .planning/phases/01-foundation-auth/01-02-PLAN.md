<task id="2.1">
## Task: Implement auth service layer

Create `backend/app/auth/services.py` with:

- `create_user(email, password)` — hash password with bcrypt, create User record, return user
- `authenticate_user(email, password)` — verify credentials, return user or None
- `create_access_token(user_id)` — JWT with 30-min expiry
- `create_refresh_token(user_id)` — JWT with 30-day expiry
- `verify_token(token)` — decode and validate JWT, return payload
- `get_user_by_email(email)` — lookup user
- `get_user_by_id(user_id)` — lookup user

JWT payload: `{"sub": user_id, "type": "access"|"refresh", "exp": timestamp}`

Password hashing: `passlib.hash.bcrypt`
JWT: `python-jose` with HS256 algorithm
</task>

<task id="2.2">
## Task: Implement auth endpoints

Create `backend/app/auth/routes.py` with these endpoints:

**POST /api/auth/register**
- Body: `{email, password}`
- Validate email format, password length >= 8
- Create user, return access + refresh tokens in httpOnly cookies
- Response: `{user: {id, email, email_verified, subscription_tier}}`

**POST /api/auth/login**
- Body: `{email, password}`
- Verify credentials, return tokens in httpOnly cookies
- Cookie config: httpOnly=true, secure=true (in prod), sameSite=Lax, path=/
- Response: `{user: {id, email, email_verified, subscription_tier}}`

**POST /api/auth/logout**
- Clear access + refresh token cookies
- Response: `{message: "Logged out"}`

**POST /api/auth/refresh**
- Read refresh token from cookie
- Validate, generate new access token
- Response: new access token cookie

**GET /api/auth/me**
- Requires valid access token
- Response: `{user: {id, email, email_verified, subscription_tier, sector_preferences}}`

**POST /api/auth/password-reset/request**
- Body: `{email}`
- Generate password reset token (short-lived, 1 hour)
- Log the reset link to console (email integration is v2)
- Response: `{message: "If email exists, reset link was sent"}`

**POST /api/auth/password-reset/confirm**
- Body: `{token, new_password}`
- Validate token, update password
- Response: `{message: "Password updated"}`

Rate limiting: 5 requests/minute on register, login, password-reset endpoints via `slowapi`.
</task>

## Task: Create auth dependency for protected routes

Create `backend/app/auth/dependencies.py`:

- `get_current_user(request)` — Extract JWT from httpOnly cookie, verify, return User
- `get_current_active_user(user)` — Ensure user exists and is active
- `get_optional_user(request)` — Same as get_current_user but returns None instead of 401

These will be used as FastAPI `Depends()` across all protected routes.

Update `backend/app/main.py` to include auth router.
</task>
