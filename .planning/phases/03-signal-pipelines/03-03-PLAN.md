<task id="3.1">
## Task: Implement LinkedIn count collection

Create `backend/app/signals/linkedin_scraper.py`:
- fetch startup `linkedin_url` (from startup profile)
- parse employee count from public company page patterns
- fallback to browser automation when HTTP response is insufficient
- return `{startup_id, employee_count, collected_at}`
- graceful fallback when count unavailable

Create parser module for robust normalization.
</task>

Ensure tasks can run independently from job-posting pipeline.
